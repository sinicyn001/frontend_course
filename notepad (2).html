HTML (язык разметки гипертекста)


HTML (от англ. HyperText Markup Language)- это язык разметки используемый для создания веб-страниц и веб-приложений.

История

HTML был разработан Тимом Бернерсом-Ли в конце 1980-х годов. Первая версия языка появилась в 1993 году.

Основные этапы развития:

  • HTML 1.0- первый стандарт (заменён более поздними версиями).
  • HTML 4.01-активно использовался до появления HTML5.
  • HTML5-современный стандарт, поддерживающий мультимедиа и семантические элементы.

    • Основные элементы HTML

    HTML состоит из различных тегов, которые создают структуру документа:

    1. Заголовки:теги (h1)-(h6).
    2. Параграфы: тег (p)
    3. Списки:маркировынные(ul) и нумерованные(ol)
    4. Гипертексты:тег (a)
